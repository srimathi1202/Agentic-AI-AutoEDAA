# Agentic AI for Automated Exploratory Data Analysis

## Project Description
This project demonstrates a simple Agentic AI system that automatically performs Exploratory Data Analysis (EDA) on a dataset.

The AI agent loads a dataset, analyzes it, detects missing values, generates statistical summaries, and creates visualizations automatically.

## Features
- Automatic dataset loading
- Dataset inspection
- Missing value detection
- Statistical analysis
- Correlation heatmap visualization
- Distribution plots
- Automated insights generation

## Technologies Used
- Python
- Pandas
- Matplotlib
- Seaborn
- GitHub

## Dataset
Titanic Dataset

## Project Workflow

Dataset  
↓  
Agent Initialization  
↓  
Data Understanding  
↓  
Statistical Analysis  
↓  
Visualization  
↓  
Insight Generation  

## Installation

Install required libraries:

pip install -r requirements.txt

## Run the Project

python autoeda_agent.py

## Output
The AI agent automatically performs exploratory data analysis and generates visual insights from the dataset.
